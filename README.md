# Audio Noise Reduction using Neural Adaptive Control and Deep Learning

## Description
This project explores different techniques for reducing noise from audio signals including LMS Filter, Wiener Filter, Neural Networks, and Deep Neural Networks.

The goal is to compare the performance of traditional signal processing methods with machine learning approaches for effective noise reduction.

## Techniques Implemented
- LMS Adaptive Filter
- Wiener Filter
- Neural Network based noise reduction
- Deep Neural Network
- Hybrid Wiener Filter + Neural Network

## Technologies Used
Python  
NumPy  
TensorFlow / Keras  
Matplotlib  

## Workflow
1. Load noisy audio signal
2. Convert signal into numerical data
3. Apply noise reduction algorithms
4. Compare filtered outputs
5. Evaluate performance

## Results
The combination of **Wiener Filter and Neural Network produced the best results**, outperforming standalone neural network and deep neural network models.
